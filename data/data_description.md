# Data Description

## 1. Refugees Data
- Source: UNHCR / Polish Government statistics
- Description: Number of Ukrainian refugees hosted in Poland by year
- Main variable:
  - refugees_total: total number of refugees

## 2. Employment Data
- Source: Eurostat / World Bank
- Description: Employment and unemployment indicators for Poland
- Main variables:
  - employment_rate_total
  - employment_rate_male
  - employment_rate_female
  - unemployment_rate

## 3. GDP Data
- Source: World Bank
- Description: Poland GDP growth and level indicators
- Main variables:
  - gdp_growth
  - gdp_current_usd
